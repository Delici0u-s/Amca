# AMCA: The Meson Build System project generator
Amca will be reworked in the near future, so idk how long this will hold up lol
## Requirements

- A properly setup c compiler that is aliased to "gcc" to the command prompt.
- the meson build system.
- python 3.x.x or newer.

Windows:
- powershell
- ```cp``` command

## Installation

Windows:
- run:
- ```INSTALL.bat```
Linux/MacOS: 
- run:
- ```chmod +x INSTALL.sh```
- ```./INSTALL.sh```

Tools that will be added: ```amca```
Automatic meson compiler (amca) will be added to your path

to inquire about how it works run ```amca --help```
